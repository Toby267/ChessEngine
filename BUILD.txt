To setup for the first time:
cd engine
mkdir build
cd build
cmake -DCMAKE_EXPORT_COMPILE_COMMANDS=ON ..

To build:
cd engine
cmake --build build

To run:
cd engine
./bin/ChessEngine

To use with https://github.com/lichess-bot-devs/lichess-bot:
setup and build
copy engine/src/ and engine/resources into lichess-bot/CurrentRelease/
copy misc/config.yml into lichess-bot/
add your lichess.org api key to config.yml
run the python script lichess-bot.py
you will have to tinker with lichess.org to create a bot account first, and generate an api key for this bot
